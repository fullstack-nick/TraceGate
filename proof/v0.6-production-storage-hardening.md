## Scope Implemented

- PostgreSQL storage backend alongside SQLite, with backend-specific migrations and shared request, replay, plugin-decision, capture, retention, and prune behavior.
- Production-mode config validation for TLS, admin bearer auth, storage caps, plugin timeout/memory/fuel, HTTPS upstreams, upstream TLS roots/CA, and unsafe target rejection.
- rustls data-plane serving plus HTTPS upstream verification using system roots and optional private CA.
- Authenticated admin routes and `POST /admin/reload` with immutable runtime setting checks and atomic active-state swap.
- Per-route concurrency semaphores, passive upstream health, and gateway rejection persistence.
- Bounded capture writer with `tracegate_capture_dropped_total` and metadata-only fallback on queue saturation.
- Local and GCP production Compose paths with internal Postgres, private CA/certs, no Cloud SQL/GKE/Redis/external managed storage, internal admin/Postgres/telemetry, and public HTTPS data plane only.
- GCP scripts updated for secret separation, TLS material generation, production deployment, HTTPS smoke, authenticated inspection, Postgres backup, and reboot-safe Docker Compose service startup.

## GCP Capture Backpressure

Final backpressure proof used a sequential HTTPS burst while a Postgres request-table lock held the capture writer. This avoided overdriving the e2-micro CPU while still filling the bounded queue.

Command shape:

```text
LOCK TABLE requests IN ACCESS EXCLUSIVE MODE;
SELECT pg_sleep(75);
```

During the lock, 1,250 production HTTPS requests were sent to the payments route with the generated CA and API key.

Observed results:

```text
image=tracegate:3d64ba469d1b
capture_dropped_before=0
requests_before=1069
captures_before=131
      2 403
     16 500
   1232 503
capture_dropped_after=231
requests_after=2094
captures_after=141
plugin_decisions_after=4017
tracegate_capture_dropped_total 231
```

The gateway continued returning bounded HTTP responses while the capture queue saturated, recorded failed gateway requests, and exposed the drop counter.
